Evită pericolele liniei de comandă posix

The command 'rm' is dangerous and its recursive force version is even more so. An alternative is to use the trash-cli package that copies files to a trashcan folder. That way complete deletion happens in two steps and mistakes might be caught. If trash-cli is not available for your system you might simply use a shell script to copy files and folders to a dedicated trash folder with a shell script.
